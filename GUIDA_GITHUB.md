# Guida rapida: pubblicazione su GitHub

## 1. Apri GitHub

Vai su GitHub e crea un nuovo repository privato.

Nome consigliato:

```text
financeplus-mandato
```

## 2. Carica i file

Carica questi file:

- `Mandato_aggiornato_report.py`
- `requirements.txt`
- `README.md`
- `.gitignore`
- `run_windows.bat`
- `run_mac_linux.sh`
- `LICENSE.txt`

Non caricare la cartella `FinancePlus_Data/`.

## 3. Avvia il programma su un PC

Scarica il repository, apri la cartella e fai doppio clic su:

```text
run_windows.bat
```

## 4. Uso consigliato

Usa sempre repository privato se lavori con dati aziendali, visure, report, PDF bancari o anagrafiche clienti.
